BotsShink v1.30 — поддержка вкладок из скриптов

Теперь приложение читает метки в верхней части BotSkripts-файлов:

Seleckt - BotSkripts
// BotSkripts-Name: ТГ-приложение
// BotSkripts-Global: true
// BotSkripts-Pinned: true
// BotSkripts-CreateTab: Телега

Что делают метки:
- BotSkripts-Global: true — скрипт общий, не привязан к одному боту.
- BotSkripts-Pinned: true — скрипт закрепляется во вкладке Скрипты.
- BotSkripts-CreateTab: Телега — после включения скрипта приложение показывает вкладку Телега.

Важно: сам обычный JS-скрипт не может нарисовать Electron-вкладку без поддержки приложения. Поэтому поддержка добавлена в приложение, а telegram_app.js является меткой/модулем для включения вкладки.
