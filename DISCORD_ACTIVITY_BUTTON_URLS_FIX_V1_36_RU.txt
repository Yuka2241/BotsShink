BotsShink 1.36.0 — исправление кликабельных кнопок Discord Activity

Что изменено:
- Discord Activity остаётся на рабочей логике подключения.
- Описание активности оставлено: Discord Owner: moz1ks / BotsShink official.
- Кнопки GitHub и Discord теперь передаются в Discord не только как labels, но и через metadata.button_urls.
- Добавлены details_url и state_url для дополнительной кликабельности строк, если Discord Desktop поддерживает это в текущей версии.
- Версия приложения остаётся 1.36.0.

Проверка:
1. Установить обновлённую сборку.
2. Открыть Discord Desktop.
3. Попросить второй аккаунт открыть профиль.
4. Нажать GitHub или Discord в активности.
