Исправление v1.34.0

Исправлено:
- Вкладка Консоль больше не пустая.
- Исправлена разметка интерфейса, из-за которой Консоль могла попасть внутрь вкладки Дискорд.
- Вкладка Дискорд теперь показывает нормальный интерфейс в стиле приложения.
- Режим Discord-бота теперь переключает поля:
  - Вкл: показываются token бота, ID сервера и ID канала.
  - Выкл: показываются Webhook и ID сервера.
- Webhook скрывается при включённом режиме бота.
- Поля бота скрываются при выключенном режиме бота.
- Discord Bot Token можно вводить как с префиксом Bot, так и без него.
