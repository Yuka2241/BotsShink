Скрипты

            
Все скрипты из папки BotSkripts. Автообновление включено.

          

          
        


        

          
Формат скрипта

          
Seleckt - BotSkripts
// BotSkripts-Name: Русское название
// BotSkripts-Config: targetPlayer
// BotSkripts-Global: true
// BotSkripts-Pinned: true
// BotSkripts-CreateTab: Телега
// BotSkripts-CreateTab: Консоль

          
Первая строка обязательна. Global/Pinned закрепляет общий скрипт во вкладке Скрипты. CreateTab говорит приложению, какую вкладку открыть после включения.

Телега

            
Настройка Telegram-бота для управления уведомлениями BotsShink.

          

          

            
            
          

        


        

          

            
Подключение Telegram

            Остановлено
          

          

            
            
          

          
Токен не показывается в логах. Chat ID нужен, чтобы приложение отвечало только тебе.

        


        

          

            
Функции Telegram

            Включённые функции появятся кнопками у твоего Telegram-бота.
          

          

            

              

                Уведомления о боте
                
Вход, выход, кик, ошибка и изменение статуса ботов.

              

              
            

            

              

                Уведомления чата
                
Отправляет в Telegram всё, что происходит в игровом чате.

              

              
            

            

              

                Онлайн сервера
                
Показывает онлайн BotsShink-ботов и текущий сервер.

              

              
            

            

              

                Игроки на сервере
                
Показывает игроков, которых видят подключённые боты.

              

              
            

          

        


        

          
Как будет работать

          
/start
Telegram покажет кнопки только тех функций, которые включены в этой вкладке.
Нажми функцию → появятся кнопки Вкл / Выкл.

        

      


      

        

          

            
Консоль

            
Управление онлайн-ботами через команды.

          

          Отключено
        


        

          

            
Команда

            Формат: say "ник" "сообщение"
          

          

            
            
          

          
say "Bot_1" "Привет"
Отправит сообщение от имени Bot_1.
say all "Привет всем"
Отправит сообщение от всех онлайн-ботов.
