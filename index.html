Скрипты

Все скрипты из папки BotSkripts. Автообновление включено.

Формат скрипта

Seleckt - BotSkripts
// BotSkripts-Name: Русское название
// BotSkripts-Config: targetPlayer
// BotSkripts-Global: true
// BotSkripts-Pinned: true
// BotSkripts-CreateTab: Телега
// BotSkripts-CreateTab: Консоль
// BotSkripts-NoBotApply: true
// BotSkripts-CreateTab: Дискорд

Первая строка обязательна. Global/Pinned закрепляет общий скрипт во вкладке Скрипты. CreateTab говорит приложению, какую вкладку открыть после включения.

Инструкция по скриптам и вкладкам

1. Базовый скрипт

Seleckt - BotSkripts
// BotSkripts-Name: Мой скрипт

module.exports = async function(api) {
  api.log('Скрипт работает');
};

Первая строка обязательна. Без неё приложение не прочитает файл как скрипт.

2. Новая вкладка из скрипта

Seleckt - BotSkripts
// BotSkripts-Name: Мой модуль
// BotSkripts-Global: true
// BotSkripts-Pinned: true
// BotSkripts-CreateTab: Моя вкладка
// BotSkripts-NoBotApply: true

Global/Pinned закрепляет модуль во вкладке “Скрипты”. CreateTab создаёт вкладку. NoBotApply запрещает назначать модуль отдельным ботам.

3. Изменение интерфейса

Через метки скрипт сообщает приложению, какую вкладку создать. Сам HTML/CSS вкладки должен быть поддержан в приложении. Для новых сложных вкладок добавляй блок в index.html, обработчики в renderer.js и IPC в main.js.

index.html → новая section tab-panel
renderer.js → кнопки, сохранение, отображение
main.js → логика, сеть, файлы, API

4. Скрипт для конкретного бота

// BotSkripts-Config: targetPlayer

Добавляет выбор игрока рядом со скриптом. Такой скрипт можно назначать каждому боту отдельно.

5. Общий модуль

Телега, Консоль и Дискорд — общие модули. Они включаются во вкладке “Скрипты” и работают для всех ботов сразу.

6. Результат

Чтобы приложение было как пластилин: маленькие действия делай обычными BotSkripts, а большие модули делай закреплёнными вкладками через Global + Pinned + CreateTab + NoBotApply.

Телега

Настройка Telegram-бота для управления уведомлениями BotsShink.

Подключение Telegram

Остановлено

Токен не показывается в логах. Chat ID нужен, чтобы приложение отвечало только тебе.

Функции Telegram

Включённые функции появятся кнопками у твоего Telegram-бота.
Уведомления о боте

Вход, выход, кик, ошибка и изменение статуса ботов.

Уведомления чата

Отправляет в Telegram всё, что происходит в игровом чате.

Онлайн сервера

Показывает онлайн BotsShink-ботов и текущий сервер.

Игроки на сервере

Показывает игроков, которых видят подключённые боты.

Как будет работать

/start
Telegram покажет кнопки только тех функций, которые включены в этой вкладке.
Нажми функцию → появятся кнопки Вкл / Выкл.

Дискорд

Интеграция Discord: режим бота или режим Webhook.

Подключение Discord

Остановлено
Discord-бот

Включено: используется токен бота, ID сервера и ID канала. Выключено: используется Webhook и ID сервера.

Режим бота: введи токен Discord-бота, ID сервера и ID канала. Webhook в этом режиме скрыт и не используется.

Функции Discord

Работают для всех ботов, когда включён общий скрипт “Дискорд”.
Уведомления о боте

Вход, выход, кик и ошибки ботов.

Уведомления чата

Отправляет игровой чат в Discord.

Онлайн сервера

Показывает текущий сервер и онлайн ботов.

Игроки на сервере

Показывает игроков, которых видят боты.
