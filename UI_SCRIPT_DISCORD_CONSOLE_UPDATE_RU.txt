BotsShink v1.34 UI Update

Добавлено:
- Кнопка "Инструкция" во вкладке "Скрипты".
- Подробный гайд по меткам BotSkripts, созданию вкладок и модулей интерфейса.
- Поддержка BotSkripts-NoBotApply: true — такие скрипты нельзя назначать отдельным ботам.
- Общий модуль "Дискорд" с вкладкой Discord.
- Режим Discord Bot Token и режим Discord Webhook.
- Вкладка "Консоль" разделена на "Общий" и "Логи".
- Логи перенесены с главной страницы в консоль.
- Список версий Minecraft теперь выбирается из выпадающего списка, auto находится сверху.
- Настройка цвета и прозрачности для кнопок или внешнего вида.

Изменено:
- Белая тема стала мягче и не такая яркая.
- Прозрачность панелей исправлена через CSS-переменную --panel-alpha.
- Точки фона теперь медленно перемещаются по всему фону.
- Кнопка очистки истории в консоли опущена ниже.
- Золотистая обводка активной вкладки остаётся убранной.
